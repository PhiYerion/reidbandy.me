Custom Arch-based Linux Distribution 

A light-weight, minimalist distribution for old Chromebooks distributed to students. This is installed via python scripts. Instead of EXT4, BTRFS is the file system with ZSTD compression to make more use of and extend the life of the 16GB eMMC storage.

Phi-AI

Very early in development. A neural network using a variety of (usually simpler) mathematical functions rather than a sigmoid. These functions along with other iteratively changing values indicate to the neural net when to 1. compactify/expand a set of nodes or 2. change the mathematical function. This additionally has the benefit of being more deterministic and thus mitigate many problems with gradient decent. 

Home Servers

A combination of mainly Arch-Linux and some debian/BSD servers. Static private IP, Hardened SSH, and automatic scripts are implemented in all servers.

Main Server: Raid-5, Terminal Management, Blockchain node, uses & can be used as a VPN, multi-user sudo configuration for 4 people, secure file transfer setup, and more.

Secure Server: Only accessible via private local network, LUKS encryption, RAID-5, and more.

Others: Cheap, simple single-purpose machines for password manager hosting, VPN (generally for more tamper-proof logging), game servers, and more. These are often to help friends.

Mining GPU Finder

During the crypto boom when GPU prices were high, it was extremely difficult to get accurate information on the prices of GPUs and tedious to find their profitability. So I made this program to figure out the actual viability of GPUs (not using MSRP) via tailored search queries to website marketplaces, string manipulation of the output, averaging the price, and comparing that to the result of another website query for real-time profitability of these cards considering electricity costs.

Additionally I have database of many of these queries publicly available at Phi-Mercuric/Phi_Database
